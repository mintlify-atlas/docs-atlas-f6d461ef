GURPS Character Sheet (GCS) is a free, open-source desktop application that lets you build, edit, and manage characters for [GURPS Fourth Edition](http://www.sjgames.com/gurps). It runs on macOS, Windows, and Linux, and supports the full breadth of GURPS 4e rules — traits, skills, spells, equipment, encumbrance, weapons, and more.

## Get started

<Steps>
  <Step title="Install GCS">
    Download the latest release for your platform from the [GitHub releases page](https://github.com/richardwilkes/gcs/releases) and run the installer.
  </Step>
  <Step title="Open or create a character sheet">
    Launch GCS, then choose **File → New Character Sheet** to start a new character, or open an existing `.gcs` sheet file.
  </Step>
  <Step title="Add traits, skills, and equipment">
    Use the Library panel on the left to drag traits, skills, spells, and equipment onto your character sheet. GCS automatically calculates point costs and derived values.
  </Step>
  <Step title="Export your character">
    When you're ready, use **File → Export** to produce a PDF or HTML version of your sheet, or use a custom text template for any output format you need.
  </Step>
</Steps>
