GURPS Character Sheet (GCS) is a standalone, interactive desktop application that lets you create, edit, and manage characters for the [GURPS Fourth Edition](http://www.sjgames.com/gurps) roleplaying game system. It runs natively on macOS, Windows, and Linux, and handles the full scope of GURPS 4e rules — from basic attributes and point accounting to traits, skills, spells, weapons, and equipment.

## What is GURPS?

GURPS (Generic Universal RolePlaying System) is a tabletop roleplaying game published by Steve Jackson Games. The Fourth Edition is a point-buy system where characters are built by spending points on attributes, advantages, skills, and spells, balanced against disadvantages and quirks. GCS fully implements these rules and performs all point calculations automatically as you build your character.

## What GCS does

GCS is built around a library-driven workflow. You maintain reusable libraries of traits, skills, spells, and equipment, then drag entries from those libraries onto character sheets. Every change immediately recalculates point totals, derived attributes, encumbrance levels, and weapon skill levels.

Key capabilities include:

- **Character creation** — Build complete GURPS 4e characters with automatic point tracking, derived attributes, and rules enforcement
- **Traits, skills, spells, and equipment** — Full support for advantages, disadvantages, quirks, skills, spells, weapons, and encumbrance
- **Library management** — Organize reusable content libraries and synchronize them from GitHub repositories
- **PDF and HTML export** — Produce print-ready character sheets in PDF or HTML, or use custom text templates for any output format
- **JavaScript scripting** — Write inline script expressions for custom prerequisites, conditional modifiers, and calculated values
- **Campaign files** — Group related sheets under a campaign to share sheet settings, attribute definitions, and body type configurations
- **Cross-platform** — Native desktop application for macOS, Windows, and Linux with full OS integration
