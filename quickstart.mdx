This guide walks you through creating a GURPS Fourth Edition character from scratch using GCS. By the end you will have a saved character sheet ready to export.

## Create your first character

<Steps>
  <Step title="Launch GCS">
    Open GCS from your Applications folder (macOS), Start menu (Windows), or by running the `gcs` binary (Linux).

    On first launch, GCS opens with the Library panel on the left and an empty workspace. If you previously saved a session, GCS restores your last workspace automatically.
  </Step>
  <Step title="Create a new character sheet">
    Choose **File → New Character Sheet** from the menu bar.

    A new sheet opens in the workspace, pre-populated with default GURPS 4e attributes and 150 starting points (the default for a standard competent character). You can change the starting point total at any time in **Edit → Sheet Settings**.
  </Step>
  <Step title="Set basic character information">
    Click on the **Identity** section at the top of the sheet to enter your character's name, player name, title, and other descriptive fields.

    Then review the **Attributes** panel and adjust any primary attributes — Strength, Dexterity, Intelligence, and Health — by clicking the up/down arrows or typing directly into the fields. GCS recalculates secondary characteristics (HP, Will, Perception, Basic Speed, Basic Move) and point costs immediately.
  </Step>
  <Step title="Add traits from the library">
    In the Library panel on the left, expand the **Traits** section to browse available advantages and disadvantages.

    Drag a trait from the library onto the **Traits** table on your character sheet. GCS adds the point cost and recalculates your total automatically. To add a disadvantage, do the same — the negative point value reduces your total.

    <Tip>
      Use the search box at the top of the Library panel to find traits by name rather than browsing the full list.
    </Tip>
  </Step>
  <Step title="Add skills">
    Expand the **Skills** section in the Library panel and drag skills onto the **Skills** table on your sheet.

    For each skill you add, click the skill row to open its editor. Set the number of points you want to invest — GCS calculates the effective level based on the controlling attribute and difficulty, following the GURPS 4e rules.
  </Step>
  <Step title="Add spells and equipment (optional)">
    If your character uses magic, drag spells from the **Spells** section of the library onto the **Spells** table. Spells work like skills — point investment determines effective level.

    To equip your character, drag items from the **Equipment** section of the library onto the **Equipment** table. GCS tracks carried weight and calculates your encumbrance level based on your ST score.
  </Step>
  <Step title="Save your character sheet">
    Choose **File → Save** (or press **Cmd+S** / **Ctrl+S**). Save the file with a `.gcs` extension.

    GCS saves all character data in a single `.gcs` file that you can reopen and edit at any time.
  </Step>
  <Step title="Export to PDF">
    Choose **File → Export → to PDF** to produce a print-ready character sheet. Select a destination and GCS renders the sheet to a PDF using the current sheet layout.

    For HTML or custom text output, use **File → Export → to HTML** or **File → Export → using Text Template** respectively.
  </Step>
</Steps>

## GCS file types

GCS uses distinct file extensions for each type of document:

| Extension | Type | Description |
|-----------|------|-------------|
| `.gcs` | Character sheet | A single GURPS character, including all attributes, traits, skills, spells, and equipment |
| `.gct` | Template | A starting-point template you can apply to new character sheets |
| `.campaign` | Campaign | Groups related character sheets and shares settings, attribute definitions, and body types |
| `.adq` | Traits library | A reusable library of advantages, disadvantages, and quirks |
| `.skl` | Skills library | A reusable library of skills |
| `.spl` | Spells library | A reusable library of spells |
| `.eqp` | Equipment library | A reusable library of equipment items |

## Command-line flags

You can pass flags when launching GCS from a terminal:

```bash
gcs [flags] [file]...
```

| Flag | Description |
|------|-------------|
| `--settings <file>` | Load settings from and store them into the specified file instead of the default location |
| `--convert` | Convert all specified files (or directories, recursively) to the current data format, then exit |
| `--sync` | Sync all `.gcs` and `.gct` files with their library sources, then exit |
| `--text <template>` | Export the specified character sheets using the given text template file, then exit |

<Note>
  `--convert` and `--sync` cannot be used together. Both flags cause GCS to exit after processing all files without opening the GUI.
</Note>
